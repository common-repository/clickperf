== Description ==

Notre plugin vous permet d’implémenter le pixel postback sur votre site Wordpress.

A la suite de l'installation d'un cookie correspondant au clickid un appel serveur est effectué sur la page de confirmation de votre choix.

Aucun prérequis, installation rapide.

Url du service :
http://www.clickperf.com

Conditions d'utilisation :
https://www.click-perf.com/terms-of-use.html

== Installation ==

Il suffit de cliquer sur le bouton "Installer" puis "Activer"
